# images/

Place your Toptails photos in this folder.

## Suggested filenames (used in index.html):
- daycare1.jpg
- daycare2.jpg
- grooming1.jpg
- boarding1.jpg
- facility1.jpg
- facility2.jpg
- og-cover.jpg  ← used for WhatsApp/Facebook link preview (1200x630px recommended)

## Tips:
- Use JPG for photos, PNG for logos
- Resize images to max 1200px wide to keep the site fast
- Free tools: squoosh.app or tinypng.com for compression
